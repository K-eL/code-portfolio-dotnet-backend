### Installation

1. Clone the repo
```sh
git clone https://github.com/K-eL/code-portfolio-dotnet-backend.git
```

2. Build the project
```sh
cd code-portfolio-dotnet-backend
dotnet build App.sln
```

3. Running the project
```sh
dotnet run --project src/App.WebApi/App.WebApi.csproj
```
